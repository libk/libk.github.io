Ubuntu GNOME的一些软件安装

发表于 2018-01-02 分类于 Linux

输入法：

  1. 安装fcitx框架
  2. 下载搜狗输入法，并安装
  3. 系统自带输入法是ibus框架，将该框架删除，输入sudo apt-get remove ibus 命令

flash

  1. 去flash官网下载安装包，然后解压
  2. 将解压之后文件夹里的libflashplayer.so移动到firefox到插件目录
  3. 在Firefox的地址栏输入about:plugins察看flash插件是否已经安装成功。

去除Firefox的标题栏

Ubuntu GNOME自带的程序标题栏很难看，又占用屏幕空间，可以用maximus扩展来隐藏。

除了在浏览器中，其他情况下标题栏的影响不大，所以，只去除浏览器中的标题栏，可以用Firefox里的htitle插件。

除此之外，可以给Firefox安装GNOME 3主题

gnome shell extensions

可以去gnome-shell插件的官网下载插件，推荐下面几个插件

  1. dash to dock ——将Dash移出应用程序概览，并将它转变为相当标准的停靠栏
  2. hide top bar ——全屏自动隐藏顶栏
  3. taskbar ——可自主设置任务栏的各种样式

gnome主题

可以去gnome look 官网去搜合适的主题。
